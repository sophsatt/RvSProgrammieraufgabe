CR Carriage Return, in Java durch \r in einem String repräsentiert. Wird genutzt um den Cursor an den Anfang der Zeile zu bewegen (z.B. im Terminal).
LF New Line Feed, in Java durch \n in einem String repräsentiert. Wird genutzt um den Cursor eine Zeile nach unten zu bewegen (z.B. im Terminal).
Dies sind besondere Zeichen die erst das Ende einer Zeile markieren und danach den beginn einer neuen Zeile ankündigen. CRLF nutzen jeweils 1 Byte (zusammen also 2 Byte).
